# 🎮 SkillArena

**SkillArena** is a fun and interactive mini-game platform designed to test and improve different cognitive and skill-based abilities.  
The project includes **unique mini games**, each focusing on a different skill area like knowledge, speed, reflexes, and memory.

---

## 🕹️ Mini Games Included

### 1️⃣ Quiz Challenge
- Test your knowledge with multiple-choice questions  
- Instant feedback on correct and wrong answers  

### 2️⃣ Typing Speed Test
- Measure your typing speed and accuracy  
- Time-based challenge  
- Helps improve typing efficiency and focus

### 3️⃣ Reaction Time Game
- Test how fast you can react to visual cues  
- Simple but addictive gameplay  
- Improves reflexes and hand-eye coordination

### 4️⃣ Memory Card Flip Game
- Flip cards to find matching pairs   
- Classic memory-boosting game experience

### 4️⃣ Canvas Player
- Controllable Player
- Collision Effect
- Gravity Control (Key - G / reset key - R)
- Custom Sprite Character
---

## 🚀 Features

- Multiple skill-based mini games in one platform  
- Simple and user-friendly interface  
- Interactive gameplay with real-time feedback  
- Responsive design (works on different screen sizes)
